# jujutsu-kaisen-qa
> Jujutsu Kaisen QA

## Install

```sh
# Create a new venv
python3 -m venv jujustu
source jujustu/bin/activate

# install dependencies
pip install -r requirements.txt
#or 
pip install langchain chromadb sentence-transformers

# run on notebook
pip install streamlit

# run app
streamlit run app.py
```
